SpotyTangoDisplay

A web app for tango DJs and teachers. Display live track info on a dancer screen — tandas, cortinas, orchestra bios, song stories — with a separate fullscreen view for the venue TV. Works with Spotify or live microphone recognition.

⬇Download for Windows (.exe)
  1. Click the button above — saves SpotyTangoDisplay-Setup.exe
  2. Double-click the EXE to install — Node.js installs automatically if needed
  3. A desktop shortcut is created → double-click it any time to launch
  4. Your browser opens the DJ control panel automatically
⚠ Windows SmartScreen warning: If Windows shows "protected your PC", click More info, then click Run anyway. The app is safe — it's just not code-signed yet.
⏳macOS app coming soon

A native macOS app is in development.

In the meantime, run from source:

  1. Install Node.js LTS
  2. Open Terminal and run:
    git clone https://github.com/GuyMayer/SpotyTangoDisplay.git
    cd SpotyTangoDisplay && node relay.js
  3. Browser opens at http://localhost:3456
